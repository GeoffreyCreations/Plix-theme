PLIX

PLIX is a the most beautiful skin for steam that modernizes the whole steam client.

A new look for the platform you already know and love. plix is fresh andexciting from the moment you open Steam.

HOW TO INSTALL
  • Navigate to steamui folder, it is usually located in "C:\Program Files (x86)\Steam\steamui"
  • In the google drive go to the web folder and choose light or dark
  • Place the libraryrootcustom.css file in the steamui folder
  • In the google drive go to FONTS folder and install all the fonts there
  • Install https://github.com/PhantomGamers/SFP
  • Launch SFP and it will apply the library skin
